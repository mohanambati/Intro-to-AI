CSCI-4450 Assignment-1 Execution Method:

Folder contains
(Solution 1.) prog1.c
(Solution 2.1) prog2_1.c
(solution 2.2) prog2_2.c

Please follow below steps to compile and execute all the Assignment-1 programs in "loki.ist.unomaha.edu" .

step 1:
compile the program by using the command below.
	gcc -std=c99 <program_name>.c
Eg: mambati@loki:~$ gcc -std=c99 prog1.c

step 2:
After successfull compilation you should be able to see "a.out" file in the same path.

step 4: Run "a.out" file using below command.
	./a.out <input_state>
Eg: mambati@loki:~$ ./a.out 431502

(**Note: Every compilation over writes "a.out" file. Please compile and test each program at a time.)
